# Python 基础语法

Python 是一种简洁易读的编程语言。

## 变量与类型

```python
# 动态类型
name = "Alice"      # str
age = 25            # int
height = 1.75       # float
is_student = True   # bool
```

<Callout type="info">
Python 是动态类型语言，变量类型在运行时确定，无需显式声明。
</Callout>

---

## 列表推导式

```python
squares = [x**2 for x in range(10)]
even_squares = [x**2 for x in range(10) if x % 2 == 0]
```

<Callout type="tip">
列表推导式是 Python 的特色语法，比传统的 for 循环更简洁高效。
</Callout>

---

## 函数定义

```python
def greet(name: str) -> str:
    """返回问候语"""
    return f"Hello, {name}!"

# Lambda 表达式
add = lambda a, b: a + b
```
