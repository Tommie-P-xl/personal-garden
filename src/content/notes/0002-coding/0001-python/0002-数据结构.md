# Python 数据结构

## 列表（List）

```python
fruits = ["apple", "banana", "cherry"]
fruits.append("date")
fruits.sort()
```

## 字典（Dictionary）

```python
person = {
    "name": "Alice",
    "age": 25,
    "city": "Beijing"
}

for key, value in person.items():
    print(f"{key}: {value}")
```

## 集合（Set）

```python
a = {1, 2, 3}
b = {3, 4, 5}
print(a & b)  # 交集: {3}
print(a | b)  # 并集: {1, 2, 3, 4, 5}
```
