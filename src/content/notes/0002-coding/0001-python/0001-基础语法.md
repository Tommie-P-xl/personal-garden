# Python 基础语法

Python 是一种简洁易读的编程语言。

## 变量与类型

```python
# 动态类型
name = "Alice"      # str
age = 25            # int
height = 1.75       # float
is_student = True   # bool
```

## 列表推导式

```python
squares = [x**2 for x in range(10)]
even_squares = [x**2 for x in range(10) if x % 2 == 0]
```

## 函数定义

```python
def greet(name: str) -> str:
    """返回问候语"""
    return f"Hello, {name}!"

# Lambda 表达式
add = lambda a, b: a + b
```
