# JavaScript 异步编程

## Promise

```javascript
const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => resolve("数据加载完成"), 1000);
  });
};

fetchData().then(data => console.log(data));
```

## async/await

```javascript
async function loadData() {
  try {
    const data = await fetchData();
    console.log(data);
  } catch (error) {
    console.error("加载失败:", error);
  }
}
```
